# Mettre CAP DAEU V2 sur Google avec GitHub Pages

## À faire sur ordinateur

1. Dézippe `CAP_DAEU_Elias_V2_GitHub_Pages.zip`.
2. Sur GitHub, crée un dépôt public nommé `cap-daeu-elias`.
3. Dans ce nouveau dépôt, choisis `Add file` puis `Upload files`.
4. Envoie **le contenu** du dossier dézippé : `index.html`, `styles.css`, `js`, `assets`, `vendor`, etc.
5. Écris `Première version CAP DAEU V2` dans la zone du commit.
6. Valide avec `Commit changes`.
7. Ouvre `Settings`, puis `Pages`.
8. Source : `Deploy from a branch`.
9. Branche : `main` ; dossier : `/(root)`.
10. Enregistre et attends quelques minutes.

Le nouveau site devrait être disponible ici :

```text
https://alive-123.github.io/cap-daeu-elias/
```

Le premier site reste ici :

```text
https://alive-123.github.io/site-sympa/
```

Les deux sites sont séparés. Les données de CAP DAEU restent sur chaque appareil ; utilise `Réglages > Exporter` pour les transférer.
